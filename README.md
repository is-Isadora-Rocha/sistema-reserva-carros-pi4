# Engenharia de Software - 

Trabalho em equipe para criação da aplicação de cadastro de reserva de carros. 

## Tecnologias e Tecnologias

- Tomcat 9 
- NodeJS 
- PostgreSQL
- Angular 10
- Eclipse e VsCode
